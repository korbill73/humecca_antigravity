# HUMECCA 고객사 관리 가이드

## 🎯 고객사를 추가/수정/삭제하는 방법

### 📝 **customers_data.js 파일 직접 편집**

고객사 목록은 `customers_data.js` 파일에 저장되어 있습니다.

#### 1. 고객사 추가하기
```javascript
const CUSTOMER_DATA = [
    { name: '기존회사1', logo: '' },
    { name: '기존회사2', logo: '' },
    // 여기에 새로운 고객사 추가
    { name: '새로운회사', logo: 'https://example.com/logo.png' },
];
