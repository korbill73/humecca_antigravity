## 🔧 트러블슈팅

### 데이터가 안 보이면?
1. Supabase SQL이 실행되었는지 확인
2. Table Editor에서 데이터 확인
3. 브라우저 콘솔(F12) 에러 확인

### Supabase 연결 에러?
1. `supabase_config.js` 확인
2. SUPABASE_URL, SUPABASE_ANON_KEY 확인
3. Supabase 대시보드에서 프로젝트 상태 확인

### 상품이 웹사이트에 안 나오면?
1. admin.html에서 상품이 등록되었는지 확인
2. active(활성화) 체크박스 확인
3. Supabase Table Editor에서 데이터 확인
