## 📞 문제 발생 시

### 브라우저 콘솔 에러가 보이면:
1. F12 > Console 탭
2. 에러 메시지 복사
3. AI에게 전달

### Supabase 연결 안 되면:
1. `supabase_config.js` 파일 확인
2. SUPABASE_URL, SUPABASE_ANON_KEY 확인
3. Supabase 대시보드에서 RLS(Row Level Security) 정책 확인

### 데이터가 안 보이면:
1. Supabase Table Editor에서 데이터 확인
2. SQL이 실행되었는지 확인
3. 브라우저 캐시 삭제 (Ctrl+Shift+Del)
