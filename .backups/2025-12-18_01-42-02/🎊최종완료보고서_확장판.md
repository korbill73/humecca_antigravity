## 🚀 사용 가이드

### Step 1: Kakao Maps API 키 설정 (선택)

1. https://developers.kakao.com 접속
2. 내 애플리케이션 > 앱 만들기
3. JavaScript 키 복사
4. `sub_company_intro.html` 214라인 수정:
```html
<!-- 변경 전 -->
<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=YOUR_KAKAO_API_KEY&libraries=services"></script>

<!-- 변경 후 -->
<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=실제키&libraries=services"></script>
```

5. "작성" 버튼
6. `sub_notice.html` 열어서 확인
