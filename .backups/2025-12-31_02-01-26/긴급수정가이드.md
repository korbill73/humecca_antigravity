# HUMECCA 관리자 시스템 - 긴급 수정 가이드

## 🚨 현재 상황
`admin.html` 파일에 JavaScript 코드가 중복되어 2615라인으로 비정상적으로 길어짐.

## ✅ 해결 방법

### Option 1: 수동 수정 (추천)
1. `admin.html` 파일 열기
2. 1780 라인부터 끝까지 스크롤
3. 중복된 `refreshHistoryList` 등의 함수 찾기
4. 1990라인 이후의 중복 코드 전체 삭제
5. 파일이 약 2000라인 정도가 되도록 정리

### Option 2: AI에게 요청
아침에 다음과 같이 요청하세요:
```
"admin.html 파일의 1787라인부터 끝까지를 확인하고, 
중복된 JavaScript 함수들을 제거해주세요.
특히 refreshHistoryList, refreshFaqList, refreshNoticeList, refreshInquiryList 함수가 
두 번 정의되어 있으면 하나만 남기고 삭제해주세요."
```

## 💡 임시 해결책
지금 당장 테스트가 필요하다면:
1. 브라우저에서 admin.html 열기
2. F12 > Console 탭 확인
3. 에러가 보이지만 기능은 대부분 작동할 것임
4. Supabase SQL 실행 후 각 탭 테스트
