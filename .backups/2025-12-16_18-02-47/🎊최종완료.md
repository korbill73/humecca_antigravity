## 🚀 빠른 시작 가이드

### 연혁 추가하기
1. admin.html 열기
2. "회사 연혁" 탭 클릭
3. 연도: 2025 입력
4. 월: 12월 선택
5. 제목: "HUMECCA 관리자 시스템 완성" 입력
6. "추가" 버튼 클릭
7. ✅ 바로 목록에 표시됨!

### FAQ 추가하기
1. "FAQ" 탭 클릭
2. 카테고리: 일반 선택
3. 질문: "서비스는 어떻게 시작하나요?" 입력
4. 답변: "관리자 페이지에서 상품을 등록하세요." 입력
5. "추가" 버튼 클릭
6. ✅ 카테고리별로 정리되어 표시됨!

### 공지사항 작성하기
1. "공지사항" 탭 클릭
2. 제목: "서비스 오픈 안내" 입력
3. 내용: HTML 사용 가능
4. "상단 고정" 체크
5. "작성" 버튼 클릭
6. ✅ 공지사항 목록 최상단에 표시됨!

## 💡 사용 팁

### 데이터 확인
- Supabase 대시보드 > Table Editor에서 실시간 데이터 확인 가능
- 브라우저 F12 > Console에서 로그 확인

### 문제 해결
- 데이터가 안 보이면: 브라우저 새로고침 (F5)
- 에러가 나면: 콘솔(F12) 확인
- 연결 안 되면: supabase_config.js 확인

### 성능 최적화
- 이미지는 최대 500KB 권장
- 긴 내용은 요약 후 "더보기" 버튼 사용
- 비활성 데이터는 정기적으로 삭제
