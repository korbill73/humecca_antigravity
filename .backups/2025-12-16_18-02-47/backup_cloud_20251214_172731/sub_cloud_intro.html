서비스 개요

Server는 안전하고 크기 조정이 가능한 컴퓨팅 인프라를 가상화하여 제공하는 서비스입니다.
웹 인터페이스나 API를 통해 필요한 서버 인프라를 간편하게 구성할 수 있습니다.
해당 서비스로 구성된 서버들의 리소스는 kt cloud에서 안전하게 관리하며, 고객 단독 사용이 보장됩니다.

서비스 특징

확장성

· Open API/SDK 제공으로 유연한 연동
· 비즈니스 확장에 따른 즉각적 자원 증설
· Incubation Center 지원

안정성

· 실시간 Monitoring 및 알림 서비스
· Load Balancer, Auto Scaling 지원
· 안전한 VM 관리 및 백업

신속성

· 수십 종의 검증된 서버 스펙/OS 제공
· 웹 콘솔을 통한 5분 내 서버 생성
· 즉시 사용 가능한 환경 제공

비용 절감

· 초기 투자비 및 유지보수 비용 0원
· 사용한 만큼 지불하는 종량제
· 저렴한 네트워크 전송 비용

고성능

· 최신 Intel CPU 및 고성능 SSD 적용
· 40Gbps 대역폭의 안정적인 백본망
· 벤치마크 상위권 성능 입증

보안

· VM별 방화벽(Firewall) 기본 제공
· 이중화된 백본망으로 재난 상황 대비
· 철저한 물리적/논리적 보안

서비스 구성 (서버 세대)

1세대 서버 구성

기존세대 Haswell CPU까지 적용되는 1세대 서버입니다. 고성능 I/O 서버 필요시 SSD Server를 신청 가능합니다.

2세대 서버 구성

Skylake, Broadwell 기반의 고성능 CPU 적용되는 2세대 서버 입니다. 기본적으로 SSD 루트 디스크가 제공되며, 데이터 디스크를 자유롭게 선택 가능합니다.

3세대 서버 구성 (New)

Cascade 기반의 고성능 CPU 적용되는 3세대 서버 입니다. Openstack플랫폼을 기반으로 고성능 VM을 제공합니다.

주요 기능

VM 구성 및 관리

  • 가상화된 VM 생성 (CPU/Disk 자유 조회)
  • 최적화된 VM 템플릿 제공

Network 제어

  • 방화벽 설정 및 포트포워딩
  • WAF 연동 보안 강화 및 LB 구성

백업과 이중화

  • 스냅샷/이미지 백업 및 복원
  • 디스크, 백본망, 서버(HA) 이중화

Monitoring

  • 로그 히스토리/트래픽 통계
  • 리소스 사용량 체크 및 알림(Watch)
