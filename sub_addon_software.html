소프트웨어 임대

시스템 운영에 필요한 Microsoft Windows Server 및 SQL Server 라이선스를 월 이용료 방식으로
합리적인 가격에 제공합니다. 기본 Core 이상은 2Core 단위로 추가됩니다.

서비스 이용 필수 안내

Core 단위 요금 산정

이용 요금은 Core 수량에 따라 산정되며,
기본 제공량 초과 시 2 Core 단위로 증설됩니다.

정책 변동 및 적용

MS 정책 변경에 따라 요금이 변동될 수 있으며,
이는 기존 이용 고객에게도 동일하게 적용됩니다.

라이선스 혼용 불가

단일 서버(VM) 내에서 임대형 라이선스와
구매형(영구) 라이선스는 혼용하여 사용할 수 없습니다.

서비스 종료 규정

호스팅/코로케이션 해지 또는 임대 기간 만료 시
라이선스 사용 권한은 즉시 자동 종료됩니다.

연체 시 이용 제한

1개월 이상 요금 미납 시 라이선스 이용이
제한되거나 중단될 수 있으니 유의하시기 바랍니다.

영구 라이선스 구매

임대 방식 외 영구 라이선스 구매를 희망하시는 경우
고객센터를 통해 별도 문의를 부탁드립니다.

도입 효과

초기 투자 비용 90% 절감

초기 하드웨어 및 소프트웨어 구매 부담을
획기적으로 줄여 재무 효율성을 극대화합니다.

복잡한 SW 단속/감사 면제

정품 라이선스 사용으로 법적 리스크와
불필요한 감사 스트레스에서 완전히 해방됩니다.

월 단위 수량 변경 및 비용 최적화

비즈니스 상황에 따라 월 단위로 유연하게
사용자 수를 조정하여 불필요한 비용을 절감합니다.

추가 비용 없이 최신 버전 즉시 사용

별도의 업데이트 비용 없이 항상 최신 기능과
보안 패치를 적용하여 경쟁력을 유지합니다.
