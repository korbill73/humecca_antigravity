소프트웨어 임대

시스템 운영에 필요한 Microsoft Windows Server 및 SQL Server 라이선스를 월 이용료 방식으로
합리적인 가격에 제공합니다. 기본 Core 이상은 2Core 단위로 추가됩니다.

주요 기능

  • Microsoft Windows Server Standard Edition
  • Microsoft Windows Server Datacenter Edition
  • Microsoft SQL Server Standard Edition
  • Microsoft SQL Server Enterprise Edition
  • 월 이용료는 Core 단위로 적용
  • 기본 Core 이상은 2Core 단위로 추가
  • Microsoft사의 정책에 의거하여 변경될 수 있음
  • 동일 서버에 임대/구매 라이선스 동시 사용 불가
  • 임대기간 완료 또는 해지시 라이선스 종료

도입 효과

1
정품 라이선스로 법적 리스크 제거
2
월 단위 임대로 초기 비용 절감
3
최신 버전 자동 업그레이드 지원
4
서버호스팅/코로케이션 연동 가능

이용 안내

  • 월 이용료는 Core 단위로 적용됩니다.
  • 기본 Core 이상은 2Core 단위로 추가됩니다.
  • 라이선스 이용요금은 Microsoft사의 정책에 의거하여 변경될 수 있으며 기존 이용 고객도 변경된 요금이 적용됩니다.
  • 동일 서버에 두 가지 형태(임대, 구매)의 라이선스를 동시에 사용할 수 없습니다.
  • 임대기간이 완료되거나 서버호스팅 또는 코로케이션을 해지하시면 라이선스 사용이 종료됩니다.
  • 라이선스 임대 비용을 1개월 이상 연체할 경우 라이선스 이용이 불가하므로, 이 점 유의하시기 바랍니다.
  • 라이선스 구매는 별도로 문의주시기 바랍니다.

* 부가세 별도 / 장기 계약 시 추가 할인 가능 / 맞춤 견적 문의 가능
