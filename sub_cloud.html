서비스 개요

Server는 안전하고 크기 조정이 가능한 컴퓨팅 인프라를 가상화하여 제공하는 서비스입니다.
웹 인터페이스나 API를 통해 필요한 서버 인프라를 간편하게 구성할 수 있습니다.
해당 서비스로 구성된 서버들의 리소스는 kt cloud에서 안전하게 관리하며, 고객 단독 사용이 보장됩니다.

서비스 특징

확장성

· Open API/SDK 제공으로 유연한 연동
· 비즈니스 확장에 따른 즉각적 자원 증설
· Incubation Center 지원

안정성

· 실시간 Monitoring 및 알림 서비스
· Load Balancer, Auto Scaling 지원
· 안전한 VM 관리 및 백업

신속성

· 수십 종의 검증된 서버 스펙/OS 제공
· 웹 콘솔을 통한 5분 내 서버 생성
· 즉시 사용 가능한 환경 제공

비용 절감

· 초기 투자비 및 유지보수 비용 0원
· 사용한 만큼 지불하는 종량제
· 저렴한 네트워크 전송 비용

고성능

· 최신 Intel CPU 및 고성능 SSD 적용
· 40Gbps 대역폭의 안정적인 백본망
· 벤치마크 상위권 성능 입증

보안

· VM별 방화벽(Firewall) 기본 제공
· 이중화된 백본망으로 재난 상황 대비
· 철저한 물리적/논리적 보안

서비스 구성 (서버 세대)

1세대 서버 구성

기존세대 Haswell CPU까지 적용되는 1세대 서버입니다. 고성능 I/O 서버 필요시 SSD Server를 신청 가능합니다.

2세대 서버 구성

Skylake, Broadwell 기반의 고성능 CPU 적용되는 2세대 서버 입니다. 기본적으로 SSD 루트 디스크가 제공되며, 데이터 디스크를 자유롭게 선택 가능합니다.

3세대 서버 구성 (New)

Cascade 기반의 고성능 CPU 적용되는 3세대 서버 입니다. Openstack플랫폼을 기반으로 고성능 VM을 제공합니다.

VDI (Virtual Desktop Infrastructure)

가상 데스크톱 인프라(VDI)는 데이터센터에서 가상 머신을 호스팅하여 사용자에게 데스크톱 환경을 제공하는 기술입니다. 언제 어디서나 동일한 업무 환경에 안전하게 접속할 수 있습니다.

강력한 보안

모든 데이터가 중앙 데이터센터에 저장되어 정보 유출 위험을 최소화합니다.

어디서나 접속

인터넷만 연결되면 어느 기기에서든 동일한 업무환경에 접속 가능합니다.

간편한 관리

중앙 집중식 관리로 소프트웨어 배포 및 업데이트가 용이합니다.

VDI 도입 효과
  • 보안 강화: 데이터가 중앙 서버에 저장되어 외부 유출 위험 최소화
  • 비용 절감: 고사양 PC 구매 없이 저사양 단말기로 업무 수행 가능
  • 업무 연속성: 재택근무, 출장 시에도 동일한 업무환경 이용
  • 유연한 확장: 필요에 따라 가상 데스크톱 추가/삭제 가능
  • 관리 효율화: OS 및 소프트웨어 중앙 관리로 IT 부서 업무 부담 감소
  • 재해 복구: 데이터센터 백업으로 신속한 업무 복구 가능
활용 사례
  • 금융기관: 민감한 금융 데이터 보호 및 컴플라이언스 준수
  • 의료기관: 환자 정보 보안 및 원격 진료 환경 구축
  • 콜센터: 다수의 상담원 PC 일괄 관리 및 보안 강화
  • 재택근무: 안전한 원격 업무환경 제공
  • 교육기관: 실습실 PC 통합 관리 및 소프트웨어 라이선스 절감

Private Cloud

Private Cloud는 기업 전용으로 구축된 클라우드 인프라로, 퍼블릭 클라우드의 유연성과 온프레미스의 보안성을 동시에 제공합니다. 민감한 데이터를 다루는 기업에 최적화된 솔루션입니다.

독립적 보안

전용 인프라로 다른 고객과 격리된 환경에서 운영됩니다.

맞춤 설정

기업 요구사항에 맞게 인프라를 자유롭게 구성할 수 있습니다.

안정적 성능

  • 규제 준수: 금융, 공공, 의료 등 규제 산업의 컴플라이언스 요구사항 충족
  • 데이터 주권: 데이터가 지정된 위치에만 저장되어 법적 요구사항 충족
